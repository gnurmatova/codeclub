CLUB

What we do in the coding club

We are strong believers in learning by doing, so in our club we focus on building small projects to learn coding and web development. The projects can be hosted on this website (check out the "Personal Pages"). In the long run, we hope to build a group of motivated kids that would represent Shelton in ACSL and CodeQuest competitions. Check out the lnguages/tools we learn on the right ->

HTML & CSS (Covered in Summer Camp)

HTML (Hyper Text Markup Language) & CSS (Cascading Style Sheets) are used all over the WWW (World Wide Web) to build the web pages. This page is built using them too!

JavaScript (Covered in Summer Camp)

Javascript is a coding/scripting language that is used to make web pages interactive. We will build cool little projects and games using JavaScript. Your Javascript projects can be hosted on this site. Check out the Gallery Section to see what projects you will be building!

Java (Not Covered in Summer Camp)

Java is one of the most popular programming languages today. This is one of the 2 programming languages we will be learning that can be used to solve problems in coding competitions.

Python (Not Covered in Summer Camp)

Python is one of the most popular programming languages today. This is one of the 2 programming languages we will be learning that can be used to solve problems in coding competitions. Python is also popular with Data Scientists who build Artificial Inelligence (AI) applications
